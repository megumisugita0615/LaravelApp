## Environment

- Docker version 20.10.21
- Docker Compose version v2.13.0
- Composer version 2.4.4
- PHP 8.1.12 (cli) 
- Laravel Framework 9.38.0
- npm 8.19.3 / node v16.18.1
- VITE v3.2.3

## Env Construction
